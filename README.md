# Lifprojet

## Setup

### Pré-requis:
- Python 3 + PIP
- NodeJS

### Instructions (Installation + Lancement)

```bash
# Pour le backend
cd backend
python3  -m .venv venv
pip -r requirements.txt install
python3 app.py # pour lancer le serveur API sur http://127.0.0.1:5000

# Pour le frontend
cd frontend
npm install
npm run dev # pour lancer le serveur de développement
```
